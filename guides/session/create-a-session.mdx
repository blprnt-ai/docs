---
title: "Create a Session"
---

<Steps>
  <Step title="Click the 'Create New Session' button">
    The button is located in the sidebar, when you expand the project you want to create a session in.

    <Frame>
      <img src="./sidebar.png" alt="New Session Sidebar" />
    </Frame>
  </Step>

  <Step title="Enter the session name">
    Give your session a name. Something meaningful and descriptive. This will only be seen by you, not the agent.

    <Note>
      Make sure to select a model unless you want to default to the [auto-router](/guides/blprnt-settings#auto-router).
    </Note>

    <Frame>
      <img src="./new-session.png" alt="New Session" />
    </Frame>
  </Step>
</Steps>
